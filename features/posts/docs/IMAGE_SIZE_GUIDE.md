# Hướng dẫn Kích thước Ảnh Bài viết

## Ảnh đại diện (Cover Image)

| Thiết bị       | Kích thước    | Tỷ lệ | Safe Area                       |
| -------------- | ------------- | ----- | ------------------------------- |
| **Desktop**    | 1200 × 630 px | 1.9:1 | 100px trái/phải, 60px trên/dưới |
| **Mobile**     | 600 × 400 px  | 1.5:1 | 40px mỗi cạnh                   |
| **Open Graph** | 1200 × 630 px | 1.9:1 | 120px trái/phải, 80px trên/dưới |

## Safe Area (Vùng an toàn)

Safe Area là vùng đảm bảo nội dung quan trọng (text, logo) luôn hiển thị đầy đủ trên mọi thiết bị và nền tảng mạng xã hội.

```
┌──────────────────────────────────┐
│          TOP MARGIN              │
│  ┌────────────────────────────┐  │
│  │                            │  │
│ L│      SAFE CONTENT AREA     │R │
│ E│                            │I │
│ F│                            │G │
│ T│                            │H │
│  │                            │T │
│  └────────────────────────────┘  │
│         BOTTOM MARGIN            │
└──────────────────────────────────┘
```

## Khuyến nghị

1. **Đặt nội dung quan trọng trong Safe Area** - Logo, tiêu đề, người trong ảnh
2. **Tránh text sát mép** - Facebook/Twitter có thể crop ảnh khác nhau
3. **Kiểm tra trên nhiều thiết bị** - Dùng Preview để xem trước
4. **Chất lượng ảnh** - Tối thiểu 300dpi cho in ấn, 72dpi cho web

## Định dạng hỗ trợ

- **JPG/JPEG** - Tốt cho ảnh chụp, kích thước nhỏ
- **PNG** - Tốt cho ảnh có text, logo, transparency
- **WebP** - Kích thước tối ưu nhất, hỗ trợ tốt trên web hiện đại

## Kích thước file khuyến nghị

- **Tối đa**: 10MB
- **Khuyến nghị**: < 2MB cho tải trang nhanh
- **Tối ưu**: Nén ảnh trước khi upload
